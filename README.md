# Convert Google Documents HTML to structured well formed HTML

## Ubuntu 16.04

### Setup

```bash
pip install -r requirements.txt
```

### Usage

```bash
python run.py
```

Open web browser at `http://127.0.0.1:9000`

## Alternative use VirtualBox + Vagrant setup

```
vagrant up
vagrant ssh
cd /vagrant
python run.py
```

Open web browser at `http://127.0.0.1:9000`
